Declarálás és Inicializálás

Ez:
    ArrayList arrayListOne;           // változó létrehozás 
    arrayListOne = new ArrayList()    // objektum létrehozás

Ugyan az, mint ez:
    ArrayList arrayListOne = new ArrayList()

Típus specifikus (String) ArrayList:
    ArrayList<String> # = new ArrayList<String>()

Iterator
    Egy objektum, saját methodokkal.
Az Enumeration egy új, továbbfejlesztett változata, aminek a segítségével más táblázatokból nyerhetünk ki adatokat.
(Enumertion = Egy interface régi adatok kinyeréséhez (Vector, Hashtable).)

Iterator létrehozása és ArrayListhez kapcsolása:
    Iterator #neve = #ArrayListneve.iterator()
    (Csak egy adatot tárol, nem array)
pl.:

    while(indivItems.hasNext()) { 
        sout(indivItems.next());
    }

Szétbontva:
    while(indivItems.hasNext()) -> a .hasNext() method addig küldi vissza a TRUE boolean értéket, amíg false jelet nem ad. (azaz elfogynak az ArrayList elemei)

    sout(indivItems.next()); -> kiírja a társított ArrayList soron következő elemét (amit az indivItems tárol).
