Collections = több kis elemnek a csoportosítása. Mérete változhat.

ArrayList<type> # = new ArrayList<>() = A collections egy fajtája. Mérete változhat. 
(Ellenben pl egy előre inicializált arrayez képest.)
(Nem használhat primitív type-ot, azaz int,char,double stb)
        
          Primitive Data Type 	  Wrapper Class
               byte 	             Byte
               short 	             Short
               int 	                 Integer
               long 	             Long
               float 	             Float
               double 	             Double
               boolean 	             Boolean
               char 	             Character
